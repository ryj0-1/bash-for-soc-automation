# Bash for SOC Automation

A collection of automation scripts and command examples designed for junior SOC analyst tasks, including log filtering, infrastructure testing, and incident triage.

## Repository Structure

* **`analiza-logow/`**: Scripts and commands for efficient network traffic filtering and analysis.
* **`skrypty-triage/`**: Automation tools for incident classification and port verification using `case` structures.
* **`automatyzacja-sieci/`**: Loop structures used for mass host connectivity scanning.

## Tech Stack
* **Scripting:** Bash / Linux Shell
* **Security Concepts:** Log parsing, incident triage, security alert logging, port analysis
